# Roadmap — Learning Speaking App

## Phase 0: Foundation (v0.1.0 – v0.3.0)
- [x] Project scaffold + TypeScript + Tailwind
- [x] Database schema + Prisma migrations
- [x] OIDC auth integration with auth.manumustudio.com

## Phase 1: Core Feature (v0.4.0 – v0.8.0)
- [x] App shell + navigation layout
- [x] Audio recording UI (MediaRecorder)
- [x] Audio upload API + R2 temp storage
- [x] Processing pipeline (QStash + Whisper + Claude)
- [ ] Results display + polling

## Phase 2: Completeness (v0.9.0 – v1.0.0)
- [ ] Session history + pagination
- [ ] GDPR compliance (delete, export, consent)
- [ ] Rate limiting + error handling + monitoring

## Phase 3: Personalization (v1.1.0+)
- [ ] Pattern trend tracking across sessions
- [ ] Context-aware feedback (LLM includes history)
- [ ] Adaptive "focus for next session" based on trends

## Phase 4: Premium (v2.0.0+)
- [ ] pgvector embeddings for transcript similarity
- [ ] RAG memory for past session context
- [ ] Weekly learning plans generated by LLM
- [ ] Custom goals and progression tracking

## Non-Goals (MVP)
- No teacher dashboard
- No gamification
- No social features
- No pronunciation scoring
- No background listening
- No mobile app / PWA
